Action Script

Sir Pierce de Mice is a game that I developed in action script. My goal was to create an entertaining game for a younger demographic that utilized voice controls. In this game, the arrow barrages can be triggered by either pressing the space bar, or the user's own voice. The louder they are, the more arrows that fly!
